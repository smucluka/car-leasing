# Preduvjeti:
 - [node.js](https://nodejs.org/en/)
 - angular cli: u cmd-u izvršiti naredbu:
 > npm install -g @angular/cli
 - [UwAmp](https://www.uwamp.com/en/?page=download) (ili xampp)
 - [SQLyog](https://github.com/webyog/sqlyog-community/wiki/Downloads) (ili neki drugi program za mysql)
 - [vscode](https://code.visualstudio.com/) (ili notepad)

# Pokretanje programa
 - nakon git clone-a u cmd-u se pozicionirati unutar foldera frontend i izvršiti naredbu (samo prvi put): 
> npm install
 - backend folder treba kopirati u C:/UwAmp/www
 - backend se pokreće tako da se upali UwAmp
 - frontend se pokreće da se u cmd-u izvrši naredba: 
> npm start
